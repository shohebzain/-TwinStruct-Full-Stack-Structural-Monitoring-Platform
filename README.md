**TwinStruct** is a modern full-stack web application that combines secure user authentication, dashboard analytics, password strength validation, interactive documentation, and an AI-powered chatbot into a single platform.

Built with **React**, **Express.js**, **Node.js**, and **Vite**, the project demonstrates modern full-stack development practices, responsive UI design, and secure authentication workflows.

# 🚀 Getting Started

## Clone the Repository

```bash id="x7jrzg"
git clone https://github.com/yourusername/TwinStruct.git
```

Navigate to the project

```bash id="9mltli"
cd TwinStruct
```

---

## Install Dependencies

```bash id="5lw7l7"
npm install
```

---

## Start the Frontend

```bash id="qjlwmh"
npm run dev
```

Frontend runs at:

```text id="es2uiv"
http://localhost:5173
```

---

## Start the Backend

```bash id="m6f5t2"
node server.cjs
```

Backend runs at:

```text id="wx0mfx"
http://localhost:5000
```
